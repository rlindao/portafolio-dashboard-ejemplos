# Dashboard Portfolio Demo

Portafolio interactivo de dashboards BI/Analítica de datos para Optima Consultoría.

## 🎯 Descripción

Aplicación web demo que presenta dashboards interactivos con datos de ejemplo, diseñada para mostrar las capacidades de BI y analítica de datos a clientes potenciales.

### Dashboards Incluidos

| Dashboard | Descripción |
|-----------|-------------|
| **📊 Ventas** | Tendencias, categorías, vendedores y distribución regional |
| **💰 Rentabilidad** | Márgenes, costos, ROI y comparativas por período |
| **👥 Clientes** | Segmentación, retención, frecuencia de compra |
| **📦 Productos** | Análisis ABC, rotación, productos estrella |
| **💵 Flujo de Caja** | Ingresos vs egresos, saldo, proyecciones |

## 🚀 Inicio rápido

```bash
npm install
npm run dev
```

## 🏗️ Build para producción

```bash
npm run build
npm run preview
```

## 📦 Deployment

El proyecto se deploya automáticamente a GitHub Pages al hacer push a la rama `main`.

## 🛠 Stack Técnico

- React 18 + Vite
- Recharts (gráficos interactivos)
- React Router (SPA)
- CSS Variables (design system dark premium)
- GitHub Actions CI/CD
